## 🗺️ 推荐学习路径

如果你是这个博客的新读者，几条按目标分类的阅读路线：

**路线 A：想做一款自己的战棋游戏**
→ C++ 基础 → 3D 数学（坐标系+矩阵部分） → 战斗公式设计 → NPC 记忆系统 → GAS 技能系统 → 2D 法线光照

**路线 B：想做高性能游戏后端/引擎**
→ 存储器 → C++ 多线程 → ECS 架构 → ECS 并行安全 → GAS 技能系统

**路线 C：想了解游戏开发全貌（在校生/转行）**
→ C++ 基础 → 3D 数学 → TCP/UDP → MMO 同步 → 战斗系统 → 随便挑 bug 记录看两个（感受真实的日常不是什么高大上的东西）
