# 关于

一个游戏客户端开发者的个人博客，记录工作中积累的 C++、Unity、渲染、网络同步、架构设计等方面的知识和经验。

不是教程——是学习笔记和工作心得。

## 博客演进

- **2016-2021** — 手动维护的 WordPress 博客
- **2021** — 迁移到自制的 TinyBlog（纯静态，GitHub Issues 做评论）
- **2024** — 文章分类扩展至 20+，覆盖 C++/Lua/Unity/ECS/渲染/方法论
- **2025-2026** — 基于两个个人项目（《归义军传说》SRPG + BattleSystem-ECS）深度补充战斗系统、ECS架构、并行安全、GAS技能系统等方向
- **2026.7** — 迁移到 VitePress

## 个人项目

- **[归义军传说](https://github.com/luoxixuan)** — 火焰纹章式 2D 像素 SRPG，归义军题材，含 NPC 记忆系统和 HD-2D 光照
- **[BattleSystem-ECS](https://github.com/luoxixuan)** — 塔防 ECS 性能基准，SOA 架构，全系统并行，10K实体 3000+ FPS

## 技术栈

- 游戏引擎：Unity、Cocos2d-x
- 语言：C++、C#、Lua、Python
- 兴趣方向：ECS架构、渲染、网络同步、AI系统、工程化
